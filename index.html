Introducción: Tecnología de Redes

La asignatura de Tecnología de Redes constituye el pilar fundamental para comprender cómo fluye la información en el mundo digital actual. Exploramos la interconexión de sistemas y la optimización de protocolos de comunicación.

Modelo OSI

Estudio integral de la comunicación de red a través de sus siete capas fundamentales.

Calculadora IP

Herramienta para el cálculo de subredes, máscaras y segmentación IPv4 de forma precisa.

Servicios de Red

Exploración de protocolos y servicios esenciales como DNS, DHCP, HTTP y seguridad SSH.

Modelo OSI

Capa 7: Aplicación (HTTP, FTP, SMTP)

Interfaz directa entre el usuario y los servicios de red.

Capa 6: Presentación

Formatea, cifra y comprime la información para su interpretación.

Capa 5: Sesión

Gestiona y controla los diálogos entre aplicaciones remotas.

Capa 4: Transporte (TCP, UDP)

Responsable de la entrega fiable de datos y control de flujo.

Capa 3: Red (IP)

Responsable del direccionamiento lógico y el enrutamiento.

Capa 2: Enlace de Datos (MAC)

Convierte los bits en tramas y asegura el acceso al medio físico.

Capa 1: Física

Transmisión física de señales a través de cables o inalámbrico.

Servicios de Red

DNS (Domain Name System)

Traduce nombres de dominio legibles en direcciones IP numéricas para la navegación.

DHCP

Asigna automáticamente direcciones IP a los dispositivos en la red de forma dinámica.

HTTP / HTTPS

Base de la transferencia de datos en la web; HTTPS añade una capa de seguridad TLS.

SSH (Secure Shell)

Permite la administración remota segura de servidores mediante canales cifrados.
